# 🧠 EchoMind

EchoMind is an AI-powered conversational assistant that delivers intelligent, context-aware responses using Large Language Models (LLMs). It leverages Retrieval-Augmented Generation (RAG) and vector search to provide accurate, relevant, and personalized answers based on user queries and stored knowledge.

---

## ✨ Features

- 🤖 Intelligent conversational interface powered by LLMs
- 🔍 Retrieval-Augmented Generation (RAG) for context-aware responses
- 📚 Semantic search using vector embeddings
- 🗂️ Pinecone Vector Database for efficient document retrieval
- ⚡ Real-time AI responses with conversation history
- 🔐 Secure authentication using JWT
- 📱 Responsive and modern user interface
- 💬 Clean chat experience with Markdown support

## 🚀 How It Works

1. User submits a query.
2. The query is converted into vector embeddings.
3. Pinecone retrieves the most relevant context using semantic search.
4. Retrieved information is combined with the user's prompt.
5. The LLM generates an accurate, context-aware response.
6. The response is displayed in a clean conversational interface.
